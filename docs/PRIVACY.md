# Privacy Policy

**Last Updated:** February 26, 2025

## Overview

TrimrPix is committed to protecting your privacy. This privacy policy explains how TrimrPix handles your data.

## Data Collection

**TrimrPix does not collect, store, or transmit any personal data.**

- ✅ No data is collected
- ✅ No analytics or tracking
- ✅ No network requests
- ✅ No third-party services
- ✅ All processing is performed locally on your device

## File Access

TrimrPix only accesses files that you explicitly:
- Drag and drop into the app
- Select through the file picker
- Add to a watched folder (if enabled)

**All files are processed locally and never leave your device.**

## Data Storage

TrimrPix stores the following data locally on your device:
- **User Settings**: Compression preferences, watch folder settings (stored in UserDefaults)
- **No images or file content** are stored permanently
- **No data** is transmitted to external servers

## Permissions

TrimrPix requires the following permissions:
- **File Access**: To read and write image files that you select
- **Downloads Folder Access**: To save optimized images (optional)

All permissions are requested explicitly and only used for the stated purpose.

## Third-Party Services

TrimrPix does not use any third-party services, analytics, or tracking tools.

## Changes to This Policy

We may update this privacy policy from time to time. Any changes will be posted on this page with an updated revision date.
